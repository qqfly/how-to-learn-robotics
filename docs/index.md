# 开源机器人学学习指南

本手册面向「非科班出身」的读者，梳理机器人学的自学路径与资料，包含先修知识、入门、进阶、具身智能等部分，每个部分都配有对应的实践。

作者：[邱强（qqfly）](https://qqfly.net)。

请从 [前言](preface.md) 开始，按下面的顺序阅读；也可以使用左侧导航。
